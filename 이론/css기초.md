# CSS의 기본

1. 스타일 시트

 스타일 시트는 크게 브라우저 기본 스타일과 사용자 스타일 두 개로 나눌 수 있다.

2. 사용자 스타일
- 인라인 스타일
- 내부 스타일 시트
- 외부 스타일 시트

3. 전체 요소에 스타일을 적용하는 전체 선택자
- *{속성: 값;}
- 전체 선택자는 웹 브라우저의 기본 스타일을 초기화할 때 자주 사용한다.

4. 특정 요소에 스타일을 적용하는 타입 선택자
- 태그명 {스타일 규칙}
- ex) p{
    font-style: italic;
}
- 타입 선택자는 특정 태그를 사용한 모든 요소에 스타일을 적용한다.
